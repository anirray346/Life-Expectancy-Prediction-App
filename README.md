# Life-Expectancy-Prediction-App
# Streamlit App

This repository contains a web application built using **Python** and **Streamlit**.  
The app provides an interactive interface for users and is deployed on **Streamlit Community Cloud**.

---

## Features
- Interactive and user-friendly web interface
- Built using Python and Streamlit
- Fast deployment on Streamlit Community Cloud
- Suitable for data analysis / machine learning applications

---

## Tech Stack
- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn

---

## To Run the App Locally

1. Clone the repository:
git clone https://github.com/anirray346/Life-Expectancy-Prediction-App.git

2. Navigate to the project directory:
cd your-repo-name

3. Install the required dependencies:
pip install -r requirements.txt

4. Run the Streamlit app:
streamlit run app.py

---


## Deployment
The application is deployed using **Streamlit Community Cloud**.  
Any updates pushed to the GitHub repository will automatically redeploy the app.

---
